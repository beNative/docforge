# DocForge v0.8.0 Release Notes

**Release date:** 2025-11-18

## Summary
DocForge v0.8.0 introduces emoji-aware titles and richer command palette coverage so power users can personalize workspaces and trigger editor actions without leaving the keyboard. The release also polishes appearance controls, search result styling, and status reporting for a smoother daily workflow.
