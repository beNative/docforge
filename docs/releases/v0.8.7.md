# Release Notes - v0.8.7

## The RAG & Clipboard Image Update

This release finalizes the RAG-enabled "Chat with Workspace" feature and introduces enhanced image support via the clipboard and drag-and-drop.

### ✨ Features

#### RAG-Powered "Chat with Workspace"
- **Configurable Similarity Threshold**: You can now tune the relevance of retrieved context in Settings. A lower threshold makes the search stricter, while a higher threshold allows for more broad context.
- **Increased Context Limit**: The maximum number of documents analyzed by the chat has been increased to **500**, enabling deep analysis of large workspaces.
- **Smart Source Attribution**: Sources used for AI responses are now streamed immediately. If the AI determines that no relevant information was found, the source list is automatically hidden to maintain a clean interface.
- **Navigation Polish**: Clicking "Create New Document" in the chat now navigates directly to the new document tab, ensuring a seamless workflow.

#### Enhanced Image Support
- **New from Clipboard (Images)**: Creating a document from the clipboard now automatically detects images and imports them as image nodes.
- **Image Pasting & Dropping**: You can now update image nodes by pasting (**Ctrl+V**) an image from your clipboard or dropping an image file directly onto the editor when an image document is open and unlocked.
- **Manual Image Type Selection**: Changing a document's language to "Image" now correctly triggers the image preview and associated interaction features.

### 🐛 Fixes

- **RAG Latency**: Optimized retrieval logic to trigger search results and response streaming in parallel.
- **Initialization Stability**: Fixed "ReferenceError" during startup by reordering chat action handlers to ensure dependencies are fully initialized.
- **Copyright Maintenance**: Updated all internal About boxes and legal notices to reflect the year **2026**.

### 🛠 Technical Changes
- Added `clipboard:read-image` IPC handler for Data URL retrieval.
- Integrated `ragSimilarityThreshold` into the `ragService` and Settings UI.
- Implemented global `paste` and `drop` event handlers in `PromptEditor` specifically for image documents.
